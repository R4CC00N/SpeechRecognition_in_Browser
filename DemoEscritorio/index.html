FUNCIONALIDAD

Para usar la aplicación, pulsa el botón 'PULSA PARA GRABAR/DETENER'. Esta aplicación está diseñada para recibir comandos de voz que se utilizarán para crear la escena.

Next Page

COMANDOS

Los siguientes comandos son estos.

* CREAR * EDITAR * ELIMINAR * GUARDAR * CARGAR * AYUDA

Prev Page Next Page

CREAR

Al decir "crear" se activa el modo creacion y en el paner de informacion apareceran los posibles objetos a crear. Durante la creacion del objeto se pueden modificar los atributos para crearlo a tu gusto. Tambien se le dara un ID para que cuente como nombre. Para salir debera crear el objeto y decir "cambio/salir"

ELIMINAR

Para eliminar un objeto tendra que decir la palabra "eliminar", esto activa el evento el cual pedira el nombre del objeto Durante la eliminacion se observara que el objeto a eliminar esta señalizado, ademas se da la opcion de cancelar la eliminacion.

GUARDAR

Este comando permite guardar la escena con todos los cambios que tenga, ademas de mantener propiedades de los objetos. Para su activacion de debera decir "guardar".

CARGAR

Este comando se utilizara para recuperar la ultima escena guardada, debido a que estas se almacenan en la cache del navegador, no se podran recuperar escenas pasadas. Para su activacion de debera decir "cargar".
